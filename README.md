# Address Search Toolbar（QGISプラグイン）

QGIS上で、属性テーブルのフィールドを使って  
**複数条件（AND / OR）で検索・選択できるプラグイン**です。

---

## 特徴

- ツールバーからダイアログを開いて検索
- 条件1・条件2の **単独 / 複合検索**
- AND / OR の意味をUI上で明示
- 数値・文字列フィールド両対応
- フリーワード検索対応
- 検索条件クリアボタン
- レイヤ未選択時の警告表示
- レイヤ情報表示  
  - ファイル形式  
  - 文字コード
- レイヤ選択の解除ボタン

## AND / OR の意味

- **AND**  
  → 条件1 **かつ** 条件2 の両方を満たす地物を検索  
  例：  
  - 市区町村 = ○○  
  - 面積 > 1000  

- **OR**  
  → 条件1 **または** 条件2 のどちらかを満たす地物を検索  

---

## 動作環境

- QGIS 3.28 以上（LTR 推奨）
- Python 3.12（QGIS 同梱）

---

## インストール方法（ZIP）

1. GitHub Releases から zip をダウンロード
2. QGIS  
   `プラグイン → プラグインの管理とインストール`
3. 「ZIPからインストール」
